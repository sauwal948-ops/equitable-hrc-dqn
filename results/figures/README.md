# Manuscript Figures
   
   This directory contains all figures from the paper "A Deep Reinforcement 
   Learning Framework for Equitable Human-Robot Collaboration: Multi-Objective 
   Task Allocation in Resource-Constrained Manufacturing"
   
   ## Figure List
   
   - **figure1_training_convergence** - DQN training over 100 episodes
   - **figure2_robustness_validation** - 20-seed statistical validation
   - **figure3_pareto_frontier** - Productivity-equity trade-off analysis
   - **figure4_cross_industry_comparison** - 5-industry performance
   - **figure5_roi_projection** - 5-year economic analysis
   - **figure6_resistance_threshold** - Worker compliance analysis
   - **figure8_monte_carlo_distributions** - 1,000-iteration validation
   - **figure9_skill_evolution** - Long-term workforce dynamics
   
   All figures are high-resolution (300 DPI minimum) for publication quality.
